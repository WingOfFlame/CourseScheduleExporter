How to use:

  1. Log into your student information system (Quest, Student Center or some other names)
  2. Go to "Academics -> Enroll -> My Class Schedule", and select the desired term
  3. Make sure the display is in "List View". Copy the whole page by pressing Ctrl+A and then Ctrl+C
  4. Paste everything into the text field by pressing Ctrl+V
  5. Click "Generate"
  6. Use the iCalendar file to import your schedule into any calendar software
